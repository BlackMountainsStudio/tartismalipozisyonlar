## SWOT Analizi

### Strengths
- Benzersiz nis: tartismali karar odakli tek AI platform
- AI otomasyon: topluluk tartismalarindan otomatik tespit
- Turkiye odakli, tamamen Turkce
- Fan katilim mekanizmasi (oylama, yorum)
- VAR derinligi (ilk karar vs nihai karar)
- Modern tech stack

### Weaknesses
- Canli skor yok
- Oyuncu istatistikleri yok
- Mobil uygulama yok
- Veri kaynagi sinirli (topluluk crawling)
- Tek lig odagi
- OpenAI API maliyet riski

### Opportunities
- VAR tartismalari global trend
- Turk futbol pazari buyuk (80M+)
- AI maliyetleri dusuyor
- Sosyal medya viralitesi (tartismali icerik)
- Yorumcu/medya ortakliklari
- Gamification potansiyeli
- PWA ile hizli mobil deneyim

### Threats
- Buyuk rakipler VAR analiz ekleyebilir
- Reddit/Eksi API politika degisiklikleri
- OpenAI fiyat/politika degisiklikleri
- TFF hassasiyeti
- Sezon disi ilgi dususu
